# Echoes of the fallen city


## 🧟 Overview
- Zombie Battle Royale Modpack is a custom modpack created for a survival battle royale experience against zombies.  
- The project includes a custom-made map, selected mods for gameplay improvements, and settings designed to create a challenging and fun survival environment.

### Features
- 🧟 **Zombie Battle Royale Gameplay**: Fight against waves of zombies in a survival-style battle.
- 🗺️ **Custom Map**: A specially made map designed for exploration and combat.
- ⚙️ **Custom Modpack**: Includes multiple mods to improve gameplay and add new mechanics.
- 🎮 **Optimized Settings**: Configured for smooth gameplay and balanced difficulty.

### Technology Used
- 💻 Minecraft mods, configuration files, and custom map tools were used to create this modpack.

### Team
- 👤 This project was built by Ken as a personal project, with the goal of creating a fun zombie battle royale experience.
